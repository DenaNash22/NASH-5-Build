# 5-Build
Module 5 Build is multi-page webpage meant to tell stories about the Indie Film Collective.
The HTML is semantic and validated. 
The CSS is responsive. 
There is no Javascript.
The website includes the following pages:
- Home (index.html)
- Film Screenings (screenings.html)
- Workshops (workshops.html)
- Collaborations (collaborations.html)
The website URL is https://denanash22.github.io/NASH-5-Build/.
